# Agent Rules

## ALWAYS DO
- work on a branch and/or worktree

## TOP LEVEL FOLDERS

- **apps** - applications by sub directory
- **bin** - scripts or binaries.  some binaries may be compiled and copied into this folder and ignored by git.
- **docs** - shared documentation
- **dot** - dotfiles
- **iac** - infrastructure as code
- **lib** - shared libraries by runtime/language
